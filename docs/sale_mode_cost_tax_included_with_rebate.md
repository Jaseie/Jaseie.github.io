# 卖车模式成本测算（含税）- 含返利 理解笔记

> 来源：用户 2026 年卖车模式成本测算（含税）- 含返利截图。当前为截图人工整理版，用于先理解口径和结构；若后续拿到 Excel/CSV/飞书导出，需要以原始表格为准替换精确数值。

## 1. 这张表在回答什么问题

这张表是在算：**卖车模式在考虑返利/让利之后，按更低的单车收入确认收入，扣除硬件成本、预计负债/后续责任成本后，还能剩多少毛利。**

它和上一张“卖车模式成本测算（含税）”结构几乎一致，但核心差异是：

```text
不含返利卖车：单车收入约 139,800
含返利卖车：单车收入约 110,442
```

也就是说，这张表是在做一个更保守的卖车口径：

```text
含返利收入
- 硬件成本
- max(运营责任成本, 预计负债)
= 单车毛利
```

## 3. 这张表的核心结论

含返利后，卖车模式仍然是正毛利，但毛利空间被大幅压缩。

对比上一张“不含返利”的卖车表：

| 口径 | 全年收入 | 全年成本 | 全年毛利 | 毛利率 |
|---|---:|---:|---:|---:|
| 卖车不含返利 | 321,540,000 | 253,798,177 | 67,741,823 | 21.07% |
| 卖车含返利 | 254,016,600 | 234,421,897 | 19,594,703 | 7.71% |
| 差异 | -67,523,400 | -19,376,280 左右 | -48,147,120 左右 | -13.36pct |

直观理解：

> 返利把单车收入从 139,800 拉低到 110,442，虽然成本也略有变化，但收入下降更大，导致全年毛利从约 6774 万降到约 1959 万。

## 4. 单车口径怎么看

含返利后的单车收入为：

```text
单车收入 = 110,442
```

表中仍然有预计负债口径：

```text
预计负债 = 单车收入 × 30% = 33,133
```

所以单车毛利可以近似理解为：

```text
单车毛利
= 110,442
- 硬件成本
- max(当期运营责任成本, 预计负债)
```

截图中单车毛利呈现三个台阶：

| 阶段 | 硬件成本 | 预计负债 | 单车毛利 |
|---|---:|---:|---:|
| 5月 | 103,165 | 33,133 | -35,475 |
| 6月 | 103,165 | 33,133 | -25,856 |
| 7-9月 | 78,535 | 33,133 | -1,226 |
| 10-12月 | 65,000 | 33,133 | 12,309 |

这说明：

> 含返利后，硬件成本即使降到 78,535，单车毛利也只是接近打平但仍略亏；只有硬件降到 65,000 后，单车毛利才明显转正。

## 5. 为什么返利影响这么大

不含返利时：

```text
单车收入 139,800
预计负债 41,940
硬件 65,000
单车毛利约 32,860
```

含返利时：

```text
单车收入 110,442
预计负债 33,133
硬件 65,000
单车毛利约 12,309
```

虽然预计负债也因为收入下降而下降：

```text
41,940 → 33,133
下降约 8,807
```

但单车收入下降更多：

```text
139,800 → 110,442
下降约 29,358
```

所以净影响大约是：

```text
收入少 29,358
- 预计负债少扣 8,807
= 单车毛利少约 20,551
```

这正好接近 10-12 月不含返利与含返利的单车毛利差：

```text
32,860 - 12,309 = 20,551
```

## 6. 成本结构拆解

表中成本结构仍然沿用卖车模式：

```text
总成本
= 硬件成本
+ max(运营责任成本, 预计负债)
```

运营责任成本继续拆成：

```text
运营单位变动成本
= 充电
+ 远驾安全员
+ 客服
+ 保险
+ 风险赔偿
+ 流量
+ 云服务费用
+ 补充运力
+ 维修费用
+ 采图制图
```

但在含返利卖车模式里，最关键的不是运营责任成本下降，而是：

```text
返利后的净售价是否还能覆盖硬件成本 + 预计负债
```

## 8. 与不含返利卖车模式对比

| 维度 | 卖车不含返利 | 卖车含返利 | 经营含义 |
|---|---:|---:|---|
| 单车收入 | 139,800 | 110,442 | 返利后净收入下降 29,358 |
| 预计负债比例 | 30% | 30% | 比例一致 |
| 预计负债金额 | 41,940 | 33,133 | 收入下降使预计负债也下降 |
| 10-12月硬件成本 | 65,000 | 65,000 | 硬件相同 |
| 10-12月单车毛利 | 32,860 | 12,309 | 单车毛利少 20,551 |
| 全年毛利 | 67,741,823 | 19,594,703 | 毛利少约 4,815 万 |

一句话：

> 返利不是“小折扣”，而是直接吞掉卖车模式大部分毛利空间。

## 9. 怎么给非财务同学解释

可以这样讲：

> 卖车原本每辆车按 13.98 万收入算，扣掉硬件和 30% 责任预留后，后期每辆能赚 3.3 万左右。加上返利后，每辆实际收入只有 11.04 万，虽然责任预留也少了，但少扣的责任成本抵不过收入下降，所以后期每辆只剩 1.2 万左右毛利。

更短：

```text
返利后的卖车模式还能赚钱，
但从高毛利变成低毛利。
核心风险是：返利一多，硬件成本稍微降不下来，就接近打平甚至亏损。
```
