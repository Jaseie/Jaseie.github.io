# 卖车模式成本测算（含税）理解笔记

> 来源：用户 2026 年卖车模式成本测算截图。当前为截图人工整理版，用于先理解口径和结构；若后续拿到 Excel/CSV/飞书导出，需要以原始表格为准替换精确数值。

## 1. 这张表在回答什么问题

这张表是在算：**卖车模式下，每个月交付一定数量的车，按单车售价确认一次性收入，同时扣除硬件成本、预计负债/后续责任成本、运营变动成本等之后，毛利和毛利率是多少。**

它和平台/租赁最大的区别是：

```text
平台/租赁：每辆车每月产生收入，成本按月摊销
卖车模式：每辆车一次性卖出，收入一次性确认，硬件成本也一次性确认
```

所以卖车模式的收入、成本、毛利量级会远大于平台/租赁模式。

## 3. 单车口径怎么看

卖车模式的单车口径是理解这张表的关键：

| 月份 | 单车收入 | 单车硬件成本 | 单车运营成本/预计责任 | 扣预计负债后毛利（约30%） | 单车毛利 |
|---|---:|---:|---:|---:|---:|
| 5月 | 139,800 | 103,165 | 42,752 | 41,940 | -6,117 |
| 6月 | 139,800 | 103,165 | 26,871 | 41,940 | -5,305 |
| 7月 | 139,800 | 78,535 | 16,337 | 41,940 | 19,325 |
| 8月 | 139,800 | 78,535 | 11,746 | 41,940 | 19,325 |
| 9月 | 139,800 | 78,535 | 8,093 | 41,940 | 19,325 |
| 10月 | 139,800 | 65,000 | 5,030 | 41,940 | 32,860 |
| 11月 | 139,800 | 65,000 | 3,079 | 41,940 | 32,860 |
| 12月 | 139,800 | 65,000 | 1,409 | 41,940 | 32,860 |

这里有一个非常重要的红色备注：

> 预计负债按收入的 30% 计算，毛利计算中运营成本低于预计负债时取预计负债值。

也就是说，卖车模式不是简单：

```text
单车毛利 = 售价 - 硬件成本 - 当期运营成本
```

而更像：

```text
单车毛利
= 单车售价
- 硬件成本
- max(当期运营责任成本, 预计负债)
```

由于预计负债是售价的 30%：

```text
139,800 × 30% = 41,940
```

所以当后续运营成本低于 41,940 时，仍然按 41,940 作为责任成本/预计负债扣除。

## 4. 卖车模式为什么后期毛利很好

最关键原因是硬件成本下降：

| 阶段 | 硬件成本 | 单车毛利 |
|---|---:|---:|
| 5-6月 | 103,165 | -6,117 / -5,305 |
| 7-9月 | 78,535 | 19,325 |
| 10-12月 | 65,000 | 32,860 |

单车售价一直是 139,800，预计负债一直是 41,940。  
所以单车毛利主要由硬件成本决定：

```text
硬件 103,165：139,800 - 103,165 - 41,940 = -5,305 左右
硬件 78,535：139,800 - 78,535 - 41,940 = 19,325
硬件 65,000：139,800 - 65,000 - 41,940 = 32,860
```

这说明：

> 卖车模式最敏感的变量是硬件成本和售价，其次是预计负债比例。

运营变动成本在后期虽然持续下降，但当它低于预计负债时，对毛利不再直接产生更大改善，因为毛利计算已经被 30% 预计负债兜底。

## 5. 成本结构拆解

卖车模式成本可理解为：

```text
单车成本
= 硬件成本
+ max(运营责任成本, 预计负债)
```

其中运营责任成本来自平台/租赁类似的运营成本结构：

```text
运营单位变动成本
= 充电
+ 远驾安全员
+ 客服
+ 保险
+ 风险赔偿
+ 流量
+ 云服务费用
+ 补充运力
+ 维修费用
+ 采图制图
```

但卖车模式中，这些成本更多是用于估算后续责任或服务成本；当它们低于预计负债时，实际毛利扣减的是预计负债。

## 8. 和平台/租赁的核心区别

| 维度 | 平台/租赁 | 卖车 |
|---|---|---|
| 收入确认 | 按月收入 | 一次性售价 |
| 收入量级 | 单车每月 2500-3000 | 单车 139,800 |
| 成本确认 | 按月摊销/运营 | 硬件一次性 + 预计负债 |
| 核心毛利驱动 | 规模、人效、运营成本 | 售价、硬件成本、预计负债 |
| 后期结果 | 接近打平 | 明显正毛利 |
| 主要风险 | 运营效率不足 | 售价下滑、硬件成本高、售后责任偏高 |

最重要的一句话：

> 卖车模式的毛利不是由月度运营效率直接决定的，而是主要由售价、硬件成本和预计负债比例决定。运营效率更多影响预计责任成本的合理性。

## 9. 怎么给非财务同学解释

可以这样说：

> 卖车模式像一次性卖出一辆车。每辆车卖 13.98 万，先扣硬件成本，再预留 30% 的未来责任成本。如果硬件还在 10.3 万，扣完就亏；如果硬件降到 7.85 万或 6.5 万，单车毛利就明显转正。

更短：

```text
卖车模式看三件事：
卖多少钱、车造多少钱、未来责任预留多少。
```

## 10. 适合做成哪些可视化

卖车模式适合重点做：

1. **单车毛利阶梯图**：5-6月亏损，7月硬件下降后转正，10月再上一个台阶。
2. **单车售价拆解瀑布图**：

```text
售价 139,800
- 硬件成本
- 预计负债 41,940
= 单车毛利
```

3. **硬件成本敏感性图**：硬件每下降 1 万，单车毛利提升 1 万。
4. **预计负债比例敏感性图**：30% 改成 25%/20% 对毛利影响多大。
5. **交付量 × 单车毛利矩阵**：看 10-12月为什么总毛利快速放大。
